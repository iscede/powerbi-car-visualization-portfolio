# Portfolio roadmap (GitHub)

## Minimum viable portfolio (2–3 weeks)

1) **Project 1 (this one): Power BI dashboard**
   - Repo with PBIX + screenshots + README narrative
   - Add a PDF export of the report (optional)

2) **Project 2: SQL + Power BI**
   - Use a public dataset
   - Show: data cleaning in SQL, a star schema, then Power BI report

3) **Project 3: Python (EDA + simple model)**
   - Notebook: EDA + baseline model + evaluation
   - Emphasis: story + business value, not complex ML

## What recruiters look for

- Clear problem statement + what you learned
- Reproducible steps (data source + how to run/open)
- Clean visuals and readable KPIs
- Consistent structure across repos

## Make your profile “visible”

- Create a **GitHub Profile README** (a repo named exactly your username)
- Pin 3 best repositories
- Add topics/tags: `powerbi`, `dax`, `sql`, `data-analysis`, `dashboard`
- Add a short bio + contact links
